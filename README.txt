# To-Do List Application

This is a simple To-Do List application. Users can add, check, edit, and delete tasks.

## Usage

1. Launch the application.
2. Click the "Add" button to add a new task.
3. Tap on a task to mark it as completed.
4. Click on the cross icon to delete a task.
5. Click on the edit button to edit a task.
6. Save tasks using local storage: Users can store added tasks in the browser.

## Requirements

- A browser is required to run this code snippet.
